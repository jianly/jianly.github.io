2019.12.13 Janely Javascript  热度 ℃

判断

条件成立，不成立。条件可以多样性

1、if / else if / else

2、三元运算符

简单if/else的特殊处理方式

条件 ？ 成立处理的事 ： 不成立处理的事

  • 如果处理的事比较多，我们括号包起来，每一件事情用逗号隔开

  • 如果不需要处理事情，可以使用null/undefined占位

3、switch case

一个变量在不同值情况下的不同操作

  • 若不加break，当前条件成立执行完成后，后面条件不论是否成立都要执行，直到break为止。(不加break可以实现变量在某些值的情况下做相同的事情)
  • 每一种case情况的比较用的都是===绝对相等

4、== VS ===

  • ==：相等。如果左右数据类型不同，是先默认转换为相同类型，然后比较

  • ===：绝对相等。如果类型不同，肯定不相等，不会默认转换数据类型

循环

重复做某些事情就是循环

1、两个关键词

  • continue: 结束当前这轮循环（后面的代码不再执行），继续执行下一轮循环

  • break: 强制结束整个循环（后面的代码不再执行），而且整个循环直接结束

2、for

  • 创建循环初始值
  • 验证循环执行条件
  • 条件成立执行循环体中的内容
  • 当前循环结束执行步长累计操作
