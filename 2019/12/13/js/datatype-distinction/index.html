JS数据类型之间的区别

2019.12.13 Janely Javascript  热度 ℃

简单案例

浏览器执行JS流程

  • 从电脑内存中分配一块内存，用来执行代码（栈内存 => stack）

  • 浏览器分配一个主线程用来自上而下执行js代码

基本类型值的存储

  • 创建变量，将变量放到当前栈内存的变量存储区域中

  • 创建一个值，将这个值放到当前栈内存值的区域中

  • =为赋值，其实赋值是让变量和值相互关联的过程

复杂值（引用类型的存储）

  • 在内存中分配一块新内存，用来存储引用类型值（堆内存 => heap）=> 内存有一个16进制地址

  • 把对象中的键值对依次存储到堆内存中

  • 把堆内存地址和变量关联起来

区别

  • 基本类型：按值操作，所以也叫作值类型

  • 引用类型：操作的是堆内存的地址（按引用地址操作的）
