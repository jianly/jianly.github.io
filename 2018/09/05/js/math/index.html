JS中的数学函数Math

2018.09.05 Janely Javascript  热度 ℃

什么是Math

数学函数，但不是函数，是一个对象，对象中存储了很多操作数字但属性方法，因此被称为函数

常用但属性和方法

.abs()

Math.abs([number value])，返回数字的绝对值（绝对值永远是正数或者零）。

.ceil() / .floor()

  • Math.ceil([number value])，向上取整
  • Math.floor([number value])，向下取整

.round()

Math.round([number value])，四舍五入

.max() / .min()

  • Math.max([x[, y[, …]]])最大值
  • Math.min([x[, y[, …]]])最小值

.sqrt() / .pow()

  • Math.sqrt(x)，平方根，负数不能开平方
  • Math.pow(x, y)，多少次幂
  • 场景：勾股定理

.random()

返回介于0和1之间的伪随机数。

获取[n~m]之间但随机整数，n < m

  • 获取1-10之间但随机整数
