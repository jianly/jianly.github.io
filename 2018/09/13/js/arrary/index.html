JS中的数组Array

2018.09.13 Janely Javascript  热度 ℃

什么是数组

数组是对象数据类型，属于特殊的对象。

常用的方法

  • 方法的作用和含义
  • 方法的实参（类型和含义）
  • 方法的返回值
  • 原来的数组是否会发生改变

1、数据增删改的方法

这部分方法都会修改原有的数组

push

  • 向数组末尾增加内容
  • 参数多个任意类型
  • 返回值是新增后的数组长度

unshift

  • 向数组开始位置添加
  • 参数多个任意类型
  • 返回值是新增后的数组长度

shift

  • 删除数组的第一项
  • 返回值是删除的那一项

pop

  • 删除数组中最后一项
  • 返回值是删除的那一项

splice

  • 数组的增加、删除、修改
  • 参数：
    • index，howmany，x 从索引index开始删除多少个元素（howmany不写，是删除到末尾），index占用删除的部分，当index为0，howmany不写时，可以清空数组
    • index，howmany，x 移除数组索引是index个位置howmany个元素，并在数组索引index位置添加新元素:
  • 返回：把删除的部分用新数组存储起来

2、数组第查询和拼接

这些方法不会改变原来第数组

slice

  • 数组的查询
  • 参数：n,m 从索引n开始，找到索引为m的地方（不包含m）
  • 返回：把找到的部分用新数组存储起来

concat

  • 数组拼接
  • 参数：多个任意值
  • 返回：拼接后的新数组，原数组不变

3、数组转化为字符串

原有数组不变

toString

  • 数组转化为字符串
  • 返回：转换后的字符串，每一项逗号分隔开

join

  • 数组转化为字符串
  • 参数：指定的分隔符，如果不指定分隔符，默认逗号
  • 返回：转换后的字符串

eval

  • 把字符串变成表达式

4、检测数组中是否包含某一项

indexOf / lastIndexOf

  • 检测当前项在数组中的第一次或者最后一次出现位置的索引值
  • 参数：要检索的这一项目内容
  • 返回：这一项出现的位置索引值（数字），如果数组中没有这一项，返回的结果是-1

5、数组的排序或者排列

reverse

  • 把数组倒过来排列
  • 返回：排列后的新数组

sort

  • 实现数组排序
  • 参数：
    • 若不传参，无法处理10以上数字排序的（默认按照每一项第一个字符排）
    • 实现多位数排序，需要传个函数，a,b => a - b 正序由小到大，b - a 倒序由大到小
  • 返回：排序后的新数组

6、遍历数组中每一项的方法

forEach

  • 遍历数组中的每一项，arr.forEach(callback(currentValue [, index [, array]])[, thisArg]);
    • currentValue 数组中正在处理的当前元素；
    • index 数组中正在处理的当前元素的索引；
    • array forEach() 方法正在操作的数组；
    • thisArg 可选参数。当执行回调函数 callback 时，用作 this 的值
  • 参数：回调函数
  • 不能使用break/continue，用return false / return true替代
