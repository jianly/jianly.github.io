JS数据类型检测方法

2018.07.25 Janely Javascript  热度 ℃

一、typeof

typeof[val]: 用来检测数据类型的运算符

  • 首先是个字符串
  • 字符串中包含对应的类型

局限性

  • typeof null => “object” 但是null并不是对象

  • 基于typeof无法细分出当前值是普通对象还是数组对象，因为只要是对象数据类型，返回的结果都是”object”

二、instanceof

用来检测当前实例是否率属于某个类

三、constructor

基于构造函数检测数据类型（也是基于某个类）

四、Object.prototype.toString.call()

检测数据类型最好的办法
