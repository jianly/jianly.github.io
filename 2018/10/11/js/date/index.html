JS中日期对象

2018.10.11 Janely Javascript  热度 ℃

基本操作

  • 获取当前客户端（本机）的时间，这个时间不能作为依据，因为可以随意更改

属性和方法

getFullYear() 获取年

getMonth() 获取月

0-11代表1月到12月

getDate() 获取日

getDay() 获取星期

0-6表示周日到周六

getHours() 获取时

getMinutes() 获取分

getSeconds() 获取秒

getMilliseconds() 获取毫秒

getTime() 获取当前日期距离 1970/1/1 00:00:00 这个日期之间的毫秒差

toLocaleDateString() 获取年月日字符串

toLocaleString() 获取完整的日期字符串

new Date() 格式化时间字符串

支持的格式

  • yyyy/mm/dd
  • yyyy/mm/dd hh:mm:ss
  • yyyy-mm-dd (ie下不支持)
