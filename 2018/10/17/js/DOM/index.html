JS中DOM及其操作

2018.10.17 Janely Javascript  热度 ℃

什么是DOM

DOM: document object model 文档对象模型，提供一些属性和方法供我们操作页面中的元素

获取DOM元素的方法

  • document.getElementById()

  • [context].getElementsByTayName() 在指定上下文中，通过标签获取一组元素集合

  • [context].getElementsByClassName() 在指定上下文中，通过样式类名获取一组元素的集合

  • document.getElementsByName() 在整个文档中，通过标签的name属性值获取一组节点集合

  • document.head / document.body / document.documentElement 获取页面中的head/body/html三个元素

  • [context].querySelector([selector]) 通过选择器获取到指定的元素对象

  • [context].querySelectorAll([selector]) 通过选择器获取到指定的元素集合

节点和描述节点之间关系的属性

节点：node 节点集合：nodeList (getElementsByName / querySelectorAll 获取所有节点的集合)

  • 元素节点

    • nodeType：1
    • nodeName: 大写的标签名
    • nodeValue: null

  • 文本节点

    • nodeType：3
    • nodeName: ‘#text’
    • nodeValue: 文本内容

  • 注释节点

    • nodeType：8
    • nodeName: ‘#commen’
    • nodeValue: 注释内容

  • 文档节点

    • nodeType：9
    • nodeName: ‘#document’
    • nodeValue: null

描述节点之间的关系

  • childNodes: 获取所有的子节点

  • children: 获取所有元素的子节点

  • firstChild: 获取第一个子节点

  • lastChild: 获取最后一个子节点

  • firstElementChild / lastElementChild: 获取第一个和最后一个元素子节点

  • previousSibling：获取上一个哥哥的节点

  • nextSibling：获取下一个弟弟的节点

  • previousElementSibling / nextElementSibling：获取哥哥和弟弟元素节点

  • parentNode: 父亲节点

动态增删改元素

  • createElement 创建元素对象
  • createTextNode 创建文本对象
  • appendChild 把元素添加到容器的末尾
  • insertBefore(新增元素，指定元素) 把元素添加到指定容器中指定元素的前面
  • cloneNode(true / false) 克隆元素或节点
  • removeChild 删除元素

自定义属性

  • 元素对象.属性名 = 属性值（原理是向元素对象对于的堆内存中添加了一个属性）
  • 基于setAttribute是把属性信息写到了元素标签的结构上。getAttribute可以把结构上存储的自定义属性值获取到
